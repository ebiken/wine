= 畑（ヴィンヤード）

Santa Lucia Highlands に所在する主な畑の位置・特徴・栽培者などを整理する。

== 開拓期を代表する畑

* **Paraiso Vineyards**: Rich と Claudia Smith が1973年頃に開拓。産地の現代ワイン栽培の先駆けの一つ
* **Sleepy Hollow Vineyard**: McFarland 家が開拓。のちに Talbott Vineyards が管理。Wine Enthusiast 誌により「グラン・クリュ」級のシャルドネ産地の一つに選ばれた。Heartwood（南）、North Crest（北）、Hounds Hollow（西）の3つの区画があり、シャルドネとピノ・ノワールのクローンが植栽されている
* **La Estancia**: Phil Johnson が開拓した畑の一つ。1970年代の産地開拓に参加

== ピノ・ノワールの評価を高めた畑

* **Pisoni Vineyard**: Gary Pisoni が1982年に植栽。サリナス渓谷を見下ろす花崗岩の多い丘陵に約40エーカー。海抜約1,300フィート。カリフォルニアを代表するピノ・ノワール畑の一つとして知られ、多くのワイナリーが果実を契約
* **Garys' Vineyard**: Gary Pisoni と Gary Franscioni の共同畑。1997年植栽、約50エーカー、ピノ・ノワールのみ。産地の中央付近に位置し、砕屑土壌・霧・冷涼な風の影響を受ける。「カリフォルニアで最も有名な畑の一つ」と評されることもある
* **Rosella's Vineyard**: Gary Franscioni が所有。自宅周辺に位置
* **Sierra Mar Vineyard**: Franscioni 家所有。産地南端の高標高に位置
* **Soberanes Vineyard**: 2008年、Pisoni と Franscioni が共同で追加した畑

== その他の著名畑

* **Double L Vineyard**: Morgan Winery が関与。Wine Enthusiast 誌の「グラン・クリュ」級シャルドネ産地の一つに選出。E&J Gallo が関連する記述もある
* **Hahn Estate**: Nicky Hahn が開拓したエステート畑

== 産地内の南北差

北側の冷涼な畑では、ピノ・ノワールが明るい赤系果実・繊細なスタイルになりやすい。南側の高標高で花崗岩質の痩せた土壌の畑では、より骨格がしっかりし、ダークフルーツとタンニンが際立つスタイルになりやすいとされる。
