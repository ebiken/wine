= 代表的なラベル・銘柄

Santa Lucia Highlands を代表するラベル・単一畑キュヴェの例を整理する。

== 単一畑・産地銘柄の例

* **Pisoni Vineyard**: 海抜約1,300フィート、花崗岩・砂岩土壌の約40エーカー（ https://www.santaluciahighlands.com/vineyard/pisoni-vineyard/ ）。色濃く凝縮したピノ・ノワールで知られ、Kosta Browne、Lucia（Pisoni 家）など複数生産者が単一畑としてボトリング
* **Garys' Vineyard**: Pisoni と Franscioni の共同畑。産地中央に位置し、ピノ・ノワールのみ。多くのワイナリーが単一畑キュヴェを生産（例: Lucia Pinot Noir Garys' Vineyard）
* **Sleepy Hollow**: シャルドネ・ピノ・ノワールの「グラン・クリュ」級畑の一つ。Talbott などが単一畑ワインを発表
* **Double L Vineyard**: Morgan が関与。シャルドネの高評価畑として知られる

== 生産者別の主な銘柄（例）

* **Kosta Browne**: Pisoni Vineyard 単一畑ピノ・ノワール、Santa Lucia Highlands 産地名ピノ・ノワール。ラズベリー・ブラックベリー・柑橘の花のアロマ、酸とタンニンのバランスが評される
* **Siduri**: Santa Lucia Highlands ピノ・ノワール（複数畑ブレンド）。Wine Enthusiast 誌で93点などの評価。黒プラム・ベイクドチェリー・ハーブ、豊かなテクスチャーとスパイスが特徴とされる
* **Pisoni Estate / Lucia / Lucy**: Pisoni 家のエステートおよび契約畑からの少量生産。ピノ・ノワール、シャルドネ、シラーなど
